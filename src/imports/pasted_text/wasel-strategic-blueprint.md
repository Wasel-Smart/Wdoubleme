Core Four-Layer Architecture (Max utilization + trust)
Intercity Ride-Sharing Layer Drivers publish trips (Amman ↔ Irbid ↔ Zarqa ↔ Aqaba ↔ Petra + return via Raje3 system). Passengers book seats → cost-sharing reduces fares 40–60%. AI matches for max occupancy.
Integrated Package Delivery Layer Drivers declare available space → packages (QR/barcode tracked) ride along passenger trips. Real-time digital proof-of-delivery + insurance bundle. Distributed logistics network at near-zero marginal cost.
Smart School Mobility Network Layer Subscription-based, safety-first system: • Student QR/NFC + facial verification boarding • Live GPS + parent push notifications & ETA • Route optimization + digital attendance • Driver background + training certification Solves driver shortages, tardiness, and parental anxiety → targets 50+ schools in first year.
Mobility Support Services Layer Additional streams: BRT feeders, elderly transport, event shuttles, municipality contracts, university campus connectors. Increases driver liquidity and platform stickiness.
