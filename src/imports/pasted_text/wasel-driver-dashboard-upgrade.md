2. Layout & Information Architecture (glanceable + prioritized)

Map integration — Full-bleed or prominent live map (Google Maps/Mapbox style) with driver location pulsing, route overlay, traffic heat. Overlay key info (pickup pin, distance, time) without cluttering.
Bottom navigation + quick actions — Persistent bottom bar: Home/Dashboard, Trips/History, Earnings/Wallet, Profile/Settings, Go Online/Offline big floating action if needed. Add context-aware floating buttons (e.g., "Accept Ride" pulsing when new request).

3. Core UX Improvements & Driver-Centric Features

AI-powered smarts (huge 2026 differentiator) — Add predictive cards: "Expected earnings next 2h: $48 (based on trends)", "Fatigue alert: 4.5h online – suggest break", "Best zone suggestion" with heatmap overlay.

Safety & trust elements — Quick SOS button always visible, share live trip link, passenger verification badge display, in-ride emergency recording indicator.

Support for larger text/dynamic type scaling.
